# 停车场管理系统简介 python738

## 项目概述

本项目是基于Python开发的停车场管理系统，采用前后端分离的技术架构，前端使用Vue，后端使用Python的Django框架，数据库采用MySQL。

## 技术栈

- 后端：Python + Django
- 前端：Vue
- 数据库：MySQL

## 系统角色

- 管理员
- 用户

## 运行环境

- 服务器环境：需配备Python运行环境及MySQL数据库
- 前端环境：需安装Node.js（建议版本高于14）

## 功能模块

详细功能模块如下：

- 注册登录模块
- 车位管理模块
- 车辆管理模块
- 收费管理模块
- 系统管理模块

## 目录结构

```
项目根目录/
│
├── backend/               # 后端代码目录
├── frontend/              # 前端代码目录
└── database/              # 数据库相关文件目录
```
